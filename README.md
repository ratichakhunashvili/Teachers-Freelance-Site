# FreeMasw - Freelancer Teachers & Students Platform

A modern web platform connecting freelancer teachers and students for personalized learning.

## Features

✅ **User Registration & Authentication**

- Register as Teacher or Student
- Secure login with email and password
- Profile customization and management

✅ **Profile Management**

- Create and customize your profile with bio, phone, location, and profile picture
- Edit profile information anytime
- Delete account if needed

✅ **Search & Discovery**

- Search for teachers and students by name or subject
- Filter by user type (Teachers/Students)
- View detailed profiles of other users
- Direct contact options

✅ **Data Persistence**

- All user data stored in JSON format using localStorage
- Data persists between sessions
- Sample data included for testing

## Files Structure

```
├── index.html      # Main HTML file with all sections
├── styles.css      # Responsive styling
├── script.js       # JavaScript functionality
├── data.json       # Sample user data
└── README.md       # This file
```

## How to Use

1. **Open the application**: Simply open `index.html` in your web browser
2. **Register**: Click "Get Started" to create a new account
3. **Login**: Use your email and password to login
4. **Search**: Use the search bar to find teachers or students by name or subject
5. **Customize Profile**: Edit your profile information anytime
6. **Contact Users**: Find and contact other users directly
